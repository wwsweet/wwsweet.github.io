lambda表达式是起到一个函数速写的作用。允许在代码内嵌入一个函数的定义。

看教程看到这一题答案发现原来有更简单的写法，用了lambda,当时不知道用法，然后百度后才明白

from functools import reduce
def prod(L):
    return reduce(lambda x,y: x*y,L)
print('3 * 5 * 7 * 9 =', prod([3, 5, 7, 9]))
